Frequently asked questions

Clear answers before
you take the next step.

Does the Book Engine write the whole book for me?

It guides the full process and generates structured prompts for each stage. Authors remain responsible for judgment, lived experience, fact checking and final editorial decisions.

Do I need an AI API key?

No. The current edition prepares copy-ready prompts and does not require a paid API connection.

Where is my project saved?

Project information is saved in the browser on the current device. Users can export a project file for backup and later import it again.

Is it only for Amazon KDP books?

The listing and launch stages are strongly suited to KDP, but the research, outlining, writing and editing stages can support many nonfiction publishing routes.

Can Hello World Publishers help me personally?

Yes. The product page links to publishing and legacy services for authors who need human guidance or done-for-you support.
